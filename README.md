## Using pipenv
- Create virtual environment : python3 -m venv venv --upgrade-deps
- pip install pipenv
- pipenv install <package-name>
- To deploy : pipenv install --deploy

## To run from docker-compose
- Open in VSCode
- Right click on docker-compose.yaml & choose "Compose Up"
